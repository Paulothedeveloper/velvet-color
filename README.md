## 🇧🇷 Português

O **VELVET** é uma ferramenta de color grading para o **DaVinci Resolve**: um **motor de cor**
que roda na GPU (transforma a imagem pixel a pixel) + um **app "cérebro"** que escolhe a câmera
de origem, aplica um look, lê os escopos e explica o porquê do grade.

**Diferente de um LUT com um slider** — o VELVET faz um processo de cor de verdade:
normaliza a origem → equilibra → tempera → finaliza. E cada operação de cor usa um **algoritmo
publicado, provado com número** (não é "no olho").

### O que já tem
- 🎥 **24 câmeras de origem** — Sony, Panasonic, ARRI, RED, Canon, BMD, Fuji, Nikon, GoPro, Apple Log, **Samsung Log**, HLG, PQ, ACES e mais.
- 🎨 **22 looks** de um clique + **8 emulações de filme**.
- 🎛️ **Motor com dezenas de controles** (modo Amador simples · Pro completo): zonal por specular, tone-map fílmico, split-tone, densidade, por-elemento (céu/água).
- 🤖 **Cérebro** — você diz a intenção, ele lê os escopos e decide o grade, explicando.
- 🌐 **5 idiomas** · Windows (Mac depois).

### Status
Em desenvolvimento ativo. Motor de cor completo e validado; app em polimento; teste final e
lançamento a caminho.

## 🇺🇸 English

**VELVET** is a color grading tool for **DaVinci Resolve**: a GPU **color engine** (pixel-level
image transform) + a **"brain" app** that picks the source camera, applies a look, reads the
scopes and explains the grade. Unlike a LUT-with-one-slider, it runs a real color process —
normalize → balance → temper → finish — and every color operation uses a **published algorithm,
proven with numbers**.

**Highlights:** 24 source cameras (incl. Samsung Log), 22 one-click looks, 8 film emulations,
a multi-control engine (Amateur/Pro), an AI brain, 5 languages. Windows first.

**Status:** active development — engine complete & validated, app in polish, launch on the way.
